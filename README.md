# DramaAuth

Authorization for [rdrama](https://rdrama.net)'s API.

Scopes:

- `identity`
- `create`
- `read`
- `update`
- `delete`
- `vote`

## Setup

    git clone https://github.com/IThinkImOKAY/DramaAuth
    cd DramaAuth
    pip3 install flask requests
    python3 app.py

The application should be running on [localhost:8000](https://localhost:8000).
